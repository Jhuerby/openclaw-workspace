## 🔧 配置中文语音

你的 Mac 目前没有中文语音包，需要下载：

### 方法一：系统设置下载（推荐）

1. 打开 **系统设置** → **辅助功能** → **朗读内容**
2. 点击 **系统语音** 下拉菜单
3. 选择 **管理语音...**
4. 找到 **中文（普通话）** 或 **Chinese (Mandarin)**
5. 点击下载按钮 ⬇️
6. 等待下载完成（约 500MB-1GB）

### 方法二：终端命令下载

```bash
# 列出所有可用语音
say -v "?"

# 查看中文语音
say -v "?" | grep -i chinese

# 常见中文语音名称：
# - Ting-Ting (大陆普通话)
# - Mei-Jia (台湾普通话)
# - Sin-Ji (粤语)
```

### 推荐中文语音

| 语音名称 | 语言 | 特点 |
|----------|------|------|
| **Ting-Ting** | 普通话（大陆） | 清晰自然，推荐 |
| **Mei-Jia** | 普通话（台湾） | 温柔女声 |
| **Sin-Ji** | 粤语 | 广东话 |

## ❓ 常见问题

### Q: 为什么没有声音？
A: 检查系统音量，确保未静音：
```bash
# 检查音量
osascript -e "output volume of (get volume settings)"

# 取消静音
osascript -e "set volume without output muted"
```

### Q: 语音太慢/太快怎么办？
A: `say` 命令不支持语速调节，但可以选择不同语音。

### Q: 可以后台播放吗？
A: 可以，在命令后加 `&`：
```bash
say "message" &
```

### Q: 支持哪些格式？
A: 仅支持文本转语音，不支持音频文件播放。
